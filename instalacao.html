Passo a Passo para Instalar o R

Siga os passos abaixo para instalar o R no seu computador:

Windows

  1. Visite o site oficial do R: CRAN.
  2. Na página inicial do CRAN, clique em "Download R for Windows".
  3. Clique em "base" para baixar a versão básica do R.
  4. Clique no link "Download R x.x.x for Windows" (onde "x.x.x" é a versão mais recente).
  5. Após o download, clique duas vezes no arquivo para iniciar a instalação.
  6. Siga as instruções do assistente de instalação, aceitando os termos e mantendo as configurações padrão.
  7. Para um ambiente de desenvolvimento mais amigável, recomendamos instalar o RStudio:
    • Visite RStudio.
    • Baixe o instalador do RStudio para Windows.
    • Execute o instalador e siga as instruções para completar a instalação.
  8. Abra o R ou o RStudio para confirmar que a instalação foi bem-sucedida.
    • No console do R, digite version e pressione Enter para verificar a versão instalada.

macOS

  1. Visite o site oficial do R: CRAN.
  2. Na página inicial do CRAN, clique em "Download R for macOS".
  3. Escolha a versão do instalador apropriada para seu sistema macOS (geralmente a mais recente).
  4. Após o download do arquivo .pkg, clique duas vezes nele para iniciar a instalação.
  5. Siga as instruções do assistente de instalação, aceitando os termos e mantendo as configurações padrão.
  6. Para um ambiente de desenvolvimento mais amigável, recomendamos instalar o RStudio:
    • Visite RStudio.
    • Baixe o instalador do RStudio para macOS.
    • Abra o arquivo .dmg e arraste o ícone do RStudio para a pasta "Aplicativos".
  7. Abra o R ou o RStudio para confirmar que a instalação foi bem-sucedida.
    • No console do R, digite version e pressione Enter para verificar a versão instalada.

Linux (Ubuntu)

  1. Abra o terminal.
  2. Digite o comando sudo apt update e pressione Enter.
  3. No terminal, digite sudo apt install r-base e pressione Enter.
  4. Siga as instruções e confirme a instalação quando solicitado.
  5. Para um ambiente de desenvolvimento mais amigável, recomendamos instalar o RStudio:
    • Baixe o pacote DEB do RStudio em RStudio.
    • No terminal, navegue até o diretório onde o pacote foi baixado (usando cd /caminho/do/diretorio).
    • Digite sudo dpkg -i rstudio-x.x.x-amd64.deb (substitua "x.x.x" pela versão do pacote) e pressione Enter.
    • Se houver dependências não resolvidas, digite sudo apt install -f e pressione Enter para corrigi-las.
  6. Abra o R ou o RStudio para confirmar que a instalação foi bem-sucedida.
    • No console do R, digite version e pressione Enter para verificar a versão instalada.

Após seguir esses passos, você estará pronto para começar a programar em R.
